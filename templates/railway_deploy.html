Deploy do Guia de Estudos no Railway

Clique no botão abaixo para implantar este aplicativo no Railway:

Deploy to Railway

Instruções após o deploy

  1. Após o deploy, acesse as variáveis de ambiente do seu projeto no Railway
  2. Adicione a variável GEMINI_API_KEY com sua chave da API do Google Gemini
  3. Adicione a variável SECRET_KEY com uma chave secreta segura para o Django
  4. Adicione a variável ALLOWED_HOSTS com seu domínio Railway (exemplo: seu-app.railway.app)
  5. Configure DEBUG=False para ambiente de produção

Variáveis de ambiente necessárias

GEMINI_API_KEY=sua_chave_api_gemini
SECRET_KEY=gere_uma_chave_secreta_segura
DEBUG=False
ALLOWED_HOSTS=seu-app.railway.app,localhost,127.0.0.1

O Railway irá automaticamente implantar seu aplicativo e configurar o banco de dados PostgreSQL.
